Getting started


Initializing a new project

The chaiproject will be a submodule in your app in the folder "lib".

  1. Add chaiproject as a submodule in your project repository:
    git submodule add git@github.com:rmehta/chaiproject lib
  2. Configure lib/py/conf.py (see conf_template.py)
  3. Run the init script:
    python lib/py/init.py admin admin123

Apache HTTP conf settings

Add the following settings to your directory or virtual host in httpd.conf file

<Directory "path/to/www">
  # allow cgi execution
  Options Indexes FollowSymLinks MultiViews ExecCGI

  # execute python files
  AddHandler cgi-script .cgi .py
</Directory>

Directory Structure

The following files will be created

  1. index_template.html: From this all static html files are generated including index.html
  2. css/style.css: Basic style elements (extension from bootstrap.css)
  3. index.html: Sample index.html that you will change

Admin User

The admin user & password is what you have specified in your init.py file

Testing

To see if your app is up, go to you browser and try to login.
