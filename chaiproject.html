Introduction

The chai project is a really simple, full stack web app framework.

Features

  1. Object store
    • GET, POST, DELETE
    • server side
    • client side
  2. User, Session and Authentication
    • users and sessions
    • cookie based authentication
  3. MVC
    • writing object classes
    • serverside events
  4. Pages
    • adding, removing, generating html
  5. Plugins
    • signin
    • forms
    • files

Sample requests

To add a new object using python requests:

requests.post('http://localhost/lib/py/objstore.py',\
     params={'type':'new_type', 'name':'n'})

Developing an app.

  1. set the framework in the "lib" folder
  2. make your basic index.html from the template (link to core)
  3. create the first user using "install.py" script
  4. make your pages
  5. generate full pages (seo friendly)
